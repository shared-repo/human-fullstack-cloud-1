1. 각 팀원별 구현 기능의 화면 만들기

   - 화면에 적용할 기술 : bootstrap5, html5, jquery, 반응형웹

   - 공통 요소 분리

   - 각 팀원의 기능별 페이지 구현 

   - 화면을 표시하기 위한 컨트롤러 구현

   - 메인 화면 또는 네비게이션에서 개별 화면으로 연결하는 링크 구현

   - 회원가입과 같이 입력이 있는 화면은 다음 두 가지 기능 모두 구현

      . 화면 표시 요청 처리

      . 입력 내용 서브밋 요청 처리 

        .. 입력 데이터를 수신하기 위한 DTO 구현 ( 설계된 데이터베이스 테이블을 참고해서 구현 - ERD에서 확인 )

        .. 컨트롤러에서 콘솔 출력을 통해 정상 동작 확인 ( 서비스 및 데이터 연동은 구현하지 않음 )
